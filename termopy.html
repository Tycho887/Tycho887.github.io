Overview

TermoPy is a Python library for simulating thermodynamic processes inside heat engines. It offers flexible tools to model cycles, plot graphs, and calculate efficiency.

Features

  • Simulates various heat engine cycles (e.g., Carnot, Rankine)
  • Calculates thermal efficiency
  • Plots PV and TS diagrams

Installation & Usage

To install the library, use pip:

pip install termopy
